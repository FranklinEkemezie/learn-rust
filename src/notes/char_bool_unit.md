# Char, Bool and Unit

## Char
A `char` is a single character of size 4 bytes. `char` must always
be written with a pair of **single** quotes.

## Bool
This is a boolean value which can be either `true` or `false`. It has
a size of 1 byte.

## Unit
This is an **empty tuple** with a size of 0 bytes. It is used to return
*nothing* in expressions or functions.
